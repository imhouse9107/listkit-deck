Technique 03

Animated Stat Counters

CSS @property integers with expo-out easing, triggered on scroll

Uses @property --stat with syntax: "<integer>" so CSS can interpolate whole numbers. The easing cubic-bezier(0.16, 1, 0.3, 1) (Expo Out) makes the numbers decelerate sharply at the end, feeling weighty and premium. Staggered delays add rhythm.

Technique 06

Noise & Grain Texture

SVG feTurbulence as data URI, applied as pseudo-element overlay

The grain is already applied globally via body::before at opacity: 0.03. For individual elements, add class grain-overlay for opacity: 0.06. The baseFrequency of 0.65-0.7 creates fine film grain. Keep it between 0.03-0.08 opacity.

Technique 07

Micro-Interactions

The easing curves, timing, and transitions that feel premium

Three rules: (1) Never use ease or linear for UI transitions, always use custom cubic-bezier. (2) Give each property its own duration. (3) Use translateY(-1px) not -4px for subtle lift.

Timing matters
