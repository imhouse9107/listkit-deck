### additionalInstructions

```
This is a high-ticket B2B sales deck ($6,000/month product). Design it like a premium venture-backed company pitch, not a marketing template.

CRITICAL DESIGN RULES:
1. Every slide must have ONE dominant visual element. Never split attention between competing elements.
2. Stats and numbers are the stars. Display them dramatically large. They should be the first thing the eye hits.
3. Use generous whitespace. Let content breathe. Crowded slides kill credibility.
4. Dark theme throughout. Deep navy/charcoal background. Electric blue and bright green as accent colors. Red only for "problem" and "cost" slides.
5. No stock photography. No clip art. If you use images, use abstract 3D renders, gradient meshes, or geometric patterns that add atmosphere without distraction.
6. Typography hierarchy must be aggressive: headlines should be 2-3x larger than body text. Stats should be the largest text on any slide.
7. Comparison slides (Slide 6 especially) must create instant visual contrast. The viewer should understand the winner in under 2 seconds.
8. The overall tone is: authoritative, data-driven, expensive-looking. Think Stripe or Linear pitch deck, not HubSpot template.
9. Keep body text minimal. Use bullet points and short phrases. Never write paragraphs on slides.
10. Slide transitions should feel smooth and modern.
```

### Recommended Theme

**Primary choice: `stratos`** (dark, premium, tech-forward)

If stratos is unavailable or does not look right, alternatives in order of preference:
- `midnight` (dark base, clean typography)
- `cosmos` (dark with gradient accents)
- Any dark theme with blue accent capability

### Recommended Image Settings

```json
{
  "imageSource": "ai",
  "imageStyle": "abstract, 3D, gradient, dark, futuristic, minimal"
}
```

Use AI-generated images only for atmospheric background elements. No stock photos of people, offices, or handshakes. If a slide does not benefit from an image, skip it entirely. Stats and data are the visual.

### Recommended Text Settings

```json
{
  "textAmount": "concise",
  "textTone": "professional, authoritative, direct"
}
```

Keep text tight. Every word must earn its place. No filler phrases like "In today's fast-paced business environment." The prospect is a CEO or VP Sales who has 7 minutes of attention. Respect that.
