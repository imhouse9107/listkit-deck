2. Animated Connecting Lines

Three styles: traveling dashes, glow pulse, and gradient flow. All pure CSS + SVG.

Style A: Traveling Dashes

CSS - TRAVELING DASHES @keyframes travelRight { 0% { stroke-dashoffset: 24; } 100% { stroke-dashoffset: 0; } } .traveling-line { stroke-dasharray: 8 16; /* 8px dash, 16px gap */ animation: travelRight 0.8s linear infinite; } SVG <svg viewBox="0 0 120 4" style="width: 120px"> <line x1="0" y1="2" x2="120" y2="2" stroke="#7c6aef" stroke-width="2" class="traveling-line"/> </svg>

Style B: Glow Pulse

CSS - GLOW PULSE @keyframes glowPulse { 0%, 100% { stroke-opacity: 0.3; } 50% { stroke-opacity: 1; } } .glow-line { animation: glowPulse 2s ease-in-out infinite; } SVG - with drop-shadow filter for neon effect <svg viewBox="0 0 200 4" style="filter: drop-shadow(0 0 4px #7c6aef);"> <line x1="0" y1="2" x2="200" y2="2" stroke="#7c6aef" stroke-width="2" class="glow-line"/> </svg>

3. CSS-Only Particle Background

Floating particles + twinkling stars. Pure CSS, no JS. Uses absolute positioning, staggered delays, and blur for depth.

CSS-only particles + stars
CSS - PARTICLE SYSTEM /* Container */ .particle-bg { position: relative; overflow: hidden; } /* Individual particle */ .particle { position: absolute; width: 3px; height: 3px; background: #7c6aef; border-radius: 50%; opacity: 0; animation: floatUp 6s ease-in infinite; } /* Glow halo */ .particle::after { content: ''; position: absolute; width: 100%; height: 100%; border-radius: 50%; background: inherit; filter: blur(4px); opacity: 0.6; } @keyframes floatUp { 0% { opacity: 0; transform: translateY(0) translateX(0) scale(0.5); } 10% { opacity: 1; } 80% { opacity: 0.6; } 100% { opacity: 0; transform: translateY(-280px) translateX(40px) scale(1.2); } } /* Stagger each particle with nth-child */ .particle:nth-child(1) { left: 5%; bottom: -10px; animation-delay: 0s; animation-duration: 5s; } .particle:nth-child(2) { left: 15%; bottom: -10px; animation-delay: 1.2s; animation-duration: 7s; width: 4px; height: 4px; } .particle:nth-child(3) { left: 25%; bottom: -10px; animation-delay: 0.6s; animation-duration: 6s; } .particle:nth-child(4) { left: 35%; bottom: -10px; animation-delay: 2.4s; animation-duration: 5.5s; width: 2px; height: 2px; } /* ... continue for 10-15 particles ... */ CSS - TWINKLING STARS .particle-star { position: absolute; width: 2px; height: 2px; background: #fff; border-radius: 50%; animation: twinkle 3s ease-in-out infinite; } @keyframes twinkle { 0%, 100% { opacity: 0.1; transform: scale(0.8); } 50% { opacity: 0.8; transform: scale(1.4); } } ACCESSIBILITY @media (prefers-reduced-motion: reduce) { .particle, .particle-star { animation: none; opacity: 0; } }

5. Animated Self-Drawing Checkmark

Circle draws first, then the checkmark path follows. Pure SVG + CSS stroke animation.

Plays once on load
Looping (green variant)
With filled background
SVG <svg width="80" height="80" viewBox="0 0 52 52"> <circle class="checkmark-circle" cx="26" cy="26" r="25" fill="none" stroke="#7c6aef" stroke-width="2"/> <path class="checkmark-check" d="M14.1 27.2l7.1 7.2 16.7-16.8" fill="none" stroke="#7c6aef" stroke-width="3" stroke-linecap="round" stroke-linejoin="round"/> </svg> CSS .checkmark-circle { stroke-dasharray: 166; /* circumference of r=25 circle */ stroke-dashoffset: 166; /* fully hidden */ animation: circleStroke 0.6s ease-in-out forwards; } .checkmark-check { stroke-dasharray: 48; /* length of checkmark path */ stroke-dashoffset: 48; /* fully hidden */ animation: checkStroke 0.3s ease-in-out 0.6s forwards; /* 0.6s delay = waits for circle */ } @keyframes circleStroke { to { stroke-dashoffset: 0; } } @keyframes checkStroke { to { stroke-dashoffset: 0; } } KEY NUMBERS - Circle circumference = 2 * PI * radius = 2 * 3.14159 * 25 = ~157 (use 166 for safety) - Checkmark path length: measure with path.getTotalLength() or estimate ~48 - Delay the checkmark animation by the circle's duration so it draws second
